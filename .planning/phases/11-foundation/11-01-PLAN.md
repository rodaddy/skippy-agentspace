  <name>Task 1: Create tools/lib/common.sh shared library</name>
  <files>tools/lib/common.sh</files>
  <action>
Create `tools/lib/` directory and `common.sh` with these exact function groups:

1. **Direct-execution guard**: Check `${BASH_SOURCE[0]} == $0` and exit 1 with error message if executed directly (not sourced).

2. **Counter initialization**: `SKIPPY_PASS=0`, `SKIPPY_WARN=0`, `SKIPPY_FAIL=0` (no _COUNT suffix per user decision).

3. **`skippy_repo_root()`**: Resolve repository root path.
   - Primary: derive from `${BASH_SOURCE[0]}` -- since file lives at `tools/lib/common.sh`, repo root is `../../` from BASH_SOURCE.
   - Validate by checking for `skills/` directory presence.
   - Fallback: check `$SKIPPY_ROOT` env var if primary fails.
   - Last resort: return unvalidated path with return code 1.

4. **Output helpers** (single argument signature per user decision):
   - `skippy_pass()` -- prints `  PASS: $1`, increments SKIPPY_PASS
   - `skippy_warn()` -- prints `  WARN: $1`, increments SKIPPY_WARN
   - `skippy_fail()` -- prints `  FAIL: $1`, increments SKIPPY_FAIL
   - `skippy_suggest()` -- prints `    Fix: $1` (no counter)

5. **`skippy_section()`**: Prints `=== $1 ===` for section headers.

6. **`skippy_summary()`**: Prints `  $SKIPPY_PASS passed, $SKIPPY_WARN warnings, $SKIPPY_FAIL failures`. Returns 0 if SKIPPY_FAIL is 0, returns 1 otherwise. Scripts end with this as their last line.

7. **`skippy_is_installed()`**: Takes skill name as $1. Returns 0 if symlink exists at `$HOME/.claude/skills/$1` OR `$HOME/.claude/commands/$1`.

Use `#!/usr/bin/env bash` shebang. Add a header comment block listing all provided functions. Do NOT add `set -euo pipefail` -- the caller's settings apply since this is sourced. Keep the file to function definitions and variable initialization only -- no command execution at source time (prevents `set -e` interaction issues from Research pitfall #3).

Claude's discretion: ANSI color codes for output, function ordering, whether to add `skippy_debug()`.
  </action>

  </verify>
