Create `tests/common-lib.bats` with ~8 tests covering common.sh functions.

Each test block must:
1. Use `setup()` that calls `load 'test_helper/common'` then `_common_setup`
2. Source common.sh: `source "$REPO_ROOT/tools/lib/common.sh"`
3. Reset counters before each test since sourcing common.sh uses `:-` defaults

Test structure:
- `@test "skippy_repo_root returns repo root"` -- call function, assert output contains "skills" dir
- `@test "skippy_pass increments counter"` -- call skippy_pass, check SKIPPY_PASS=1
- `@test "skippy_warn increments counter"` -- call skippy_warn, check SKIPPY_WARN=1

- `@test "skippy_summary returns 0 with no failures"` -- run skippy_summary, assert_success
- `@test "skippy_summary returns 1 with failures"` -- set SKIPPY_FAIL=1, run skippy_summary, assert_failure
- `@test "skippy_is_installed detects symlink"` -- create symlink in sandboxed HOME, assert function returns 0
- `@test "skippy_is_installed returns 1 for missing skill"` -- assert function returns 1 for nonexistent skill

Important: Reset counters (SKIPPY_PASS=0, SKIPPY_WARN=0, SKIPPY_FAIL=0) BEFORE sourcing common.sh since common.sh uses `${SKIPPY_PASS:-0}` which won't reset if already set from a prior test.
