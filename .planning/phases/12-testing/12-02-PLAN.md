install.sh modes:
- No args: show_status (prints table, exit 0)
- --core: install_skill "core" only
- --all: iterate skills/*, install each
- positional args: install named skills
- --target=skills|commands|auto: override target directory

uninstall.sh modes:
- No args: show usage + list_installed
- --all: ONLY removes symlinks pointing into REPO_ROOT/skills/ (readlink safety)
- positional args: uninstall named skills
- validate_skill_name: same validation as install

Critical safety: uninstall.sh --all checks `readlink "$link"` against `$REPO_SKILLS_DIR/*` before removing.
