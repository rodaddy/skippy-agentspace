# Dark Hacker Palette

Dark backgrounds with neon accents. Terminal/cyberpunk aesthetic. Great for dev tooling, CI/CD pipelines, security diagrams.

## Canvas
| Property | Value |
|----------|-------|
| Background | `#0d1117` |

## Shape Colors (Semantic)

| Purpose | Fill | Stroke |
|---------|------|--------|
| Primary/Neutral | `#1f2937` | `#00ff88` |
| Secondary | `#1a2332` | `#00d4ff` |
| Tertiary | `#172130` | `#8b5cf6` |
| Start/Trigger | `#1a2e1a` | `#00ff88` |
| End/Success | `#0d2818` | `#22c55e` |
| Warning/Reset | `#2d1b1b` | `#ff4444` |
| Decision | `#2d2a1b` | `#fbbf24` |
| AI/LLM | `#1e1a2e` | `#a78bfa` |
| Inactive/Disabled | `#1a1a2e` | `#4b5563` (dashed stroke) |
| Error | `#2d1010` | `#ef4444` |

## Text Colors

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#00ff88` | Section headings, major labels |
| Subtitle | `#00d4ff` | Subheadings, secondary labels |
| Body/Detail | `#9ca3af` | Descriptions, annotations |
| On dark fills | `#e5e7eb` | Text inside dark shapes |
| Accent | `#ff6bcb` | Highlights, important callouts |

## Evidence Artifacts

| Artifact | Background | Text Color |
|----------|-----------|------------|
| Code snippet | `#000000` | `#00ff88` (green on black) |
| JSON/data | `#000000` | `#00d4ff` (cyan on black) |

## Lines & Arrows

| Element | Color |
|---------|-------|
| Arrows | `#00ff88` or match source stroke |
| Structural lines | `#374151` |
| Marker dots | Fill + stroke: `#00ff88` |
