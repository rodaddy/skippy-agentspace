# Pastels Palette

Soft, muted pastel colors. Friendly and approachable. Good for brainstorming, concept maps, educational diagrams, user-facing docs.

## Canvas
| Property | Value |
|----------|-------|
| Background | `#fefefe` |

## Shape Colors (Semantic)

| Purpose | Fill | Stroke |
|---------|------|--------|
| Primary/Neutral | `#bfdbfe` | `#3b82f6` |
| Secondary | `#c7d2fe` | `#6366f1` |
| Tertiary | `#ddd6fe` | `#8b5cf6` |
| Start/Trigger | `#bbf7d0` | `#16a34a` |
| End/Success | `#a7f3d0` | `#059669` |
| Warning/Reset | `#fecdd3` | `#e11d48` |
| Decision | `#fde68a` | `#ca8a04` |
| AI/LLM | `#e9d5ff` | `#9333ea` |
| Inactive/Disabled | `#f1f5f9` | `#94a3b8` (dashed stroke) |
| Error | `#fecaca` | `#dc2626` |

## Text Colors

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#1e293b` | Section headings, major labels |
| Subtitle | `#475569` | Subheadings, secondary labels |
| Body/Detail | `#64748b` | Descriptions, annotations |
| On light fills | `#334155` | Text inside pastel shapes |
| On dark fills | `#f8fafc` | Text inside darker elements |

## Evidence Artifacts

| Artifact | Background | Text Color |
|----------|-----------|------------|
| Code snippet | `#1e293b` | `#a5b4fc` (light indigo) |
| JSON/data | `#1e293b` | `#86efac` (light green) |

## Lines & Arrows

| Element | Color |
|---------|-------|
| Arrows | `#6366f1` or match source stroke |
| Structural lines | `#94a3b8` |
| Marker dots | Fill + stroke: `#6366f1` |
