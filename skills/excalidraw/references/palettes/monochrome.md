# Monochrome Palette

Grayscale with one accent color (blue). Minimal, print-friendly, high contrast. Good for documentation, whitepapers, formal diagrams.

## Canvas
| Property | Value |
|----------|-------|
| Background | `#ffffff` |

## Shape Colors (Semantic)

| Purpose | Fill | Stroke |
|---------|------|--------|
| Primary/Neutral | `#e5e7eb` | `#1f2937` |
| Secondary | `#f3f4f6` | `#374151` |
| Tertiary | `#f9fafb` | `#4b5563` |
| Start/Trigger | `#dbeafe` | `#1e40af` |
| End/Success | `#d1d5db` | `#111827` |
| Warning/Reset | `#fef2f2` | `#991b1b` |
| Decision | `#f3f4f6` | `#1f2937` (2px stroke) |
| AI/LLM | `#ede9fe` | `#5b21b6` |
| Inactive/Disabled | `#f9fafb` | `#9ca3af` (dashed stroke) |
| Error | `#fee2e2` | `#b91c1c` |
| **Accent** | `#3b82f6` | `#1e40af` |

## Text Colors

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#111827` | Section headings, major labels |
| Subtitle | `#374151` | Subheadings, secondary labels |
| Body/Detail | `#6b7280` | Descriptions, annotations |
| On light fills | `#1f2937` | Text inside light shapes |
| On dark fills | `#ffffff` | Text inside dark shapes |
| Accent | `#1e40af` | Key highlights, focal elements |

## Evidence Artifacts

| Artifact | Background | Text Color |
|----------|-----------|------------|
| Code snippet | `#111827` | `#e5e7eb` (light gray on near-black) |
| JSON/data | `#111827` | `#93c5fd` (light blue on near-black) |

## Lines & Arrows

| Element | Color |
|---------|-------|
| Arrows | `#374151` or accent `#1e40af` for emphasis |
| Structural lines | `#4b5563` |
| Marker dots | Fill + stroke: `#374151` |
