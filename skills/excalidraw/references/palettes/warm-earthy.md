# Warm Earthy Palette

Amber, terracotta, sage tones. Organic, approachable feel. Good for product flows, user journeys, stakeholder presentations.

## Canvas
| Property | Value |
|----------|-------|
| Background | `#faf7f2` |

## Text Colors

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#78350f` | Section headings, major labels |
| Subtitle | `#92400e` | Subheadings, secondary labels |
| Body/Detail | `#78716c` | Descriptions, annotations |
| On light fills | `#44403c` | Text inside light shapes |
| On dark fills | `#fef3c7` | Text inside dark shapes |
