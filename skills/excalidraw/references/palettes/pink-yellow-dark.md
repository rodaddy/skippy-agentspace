# Pink & Yellow on Dark Palette

Hot pinks and bright yellows against dark backgrounds. Bold, energetic, eye-catching. Good for creative projects, marketing diagrams, standout presentations.

## Canvas
| Property | Value |
|----------|-------|
| Background | `#1a1a2e` |

## Shape Colors (Semantic)

| Purpose | Fill | Stroke |
|---------|------|--------|
| Primary/Neutral | `#2d1b4e` | `#f472b6` |
| Secondary | `#1e1b3a` | `#fbbf24` |
| Tertiary | `#2a1f3d` | `#fb923c` |
| Start/Trigger | `#1a2e1a` | `#fbbf24` |
| End/Success | `#1a2e2a` | `#34d399` |
| Warning/Reset | `#3d1a1a` | `#f87171` |
| Decision | `#2d2a1b` | `#fde047` |
| AI/LLM | `#2e1a3d` | `#e879f9` |
| Inactive/Disabled | `#1f1f2e` | `#6b7280` (dashed stroke) |
| Error | `#3d1010` | `#ef4444` |

## Text Colors

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#f472b6` | Section headings, major labels |
| Subtitle | `#fbbf24` | Subheadings, secondary labels |
| Body/Detail | `#a1a1aa` | Descriptions, annotations |
| On dark fills | `#f5f5f5` | Text inside dark shapes |
| Pink accent | `#ec4899` | Hot highlights |
| Yellow accent | `#facc15` | Warm highlights |

## Evidence Artifacts

| Artifact | Background | Text Color |
|----------|-----------|------------|
| Code snippet | `#0f0f1a` | `#f472b6` (pink on near-black) |
| JSON/data | `#0f0f1a` | `#fbbf24` (yellow on near-black) |

## Lines & Arrows

| Element | Color |
|---------|-------|
| Arrows | `#f472b6` or `#fbbf24` |
| Structural lines | `#4a4a5a` |
| Marker dots | Fill: `#f472b6`, stroke: `#ec4899` |
